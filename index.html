Q1: What is ZoneOut about?
ZoneOut is a YSWS where you create something themed around horror, mystery or ARG!
Q2: What is the theme?
Horror, mystery and ARG focusing on immersiveness and storytelling. [and, Of course good art!]
Q3: What kind of projects are to be submitted?
Projects can be:
• A website
• A terminal / CLI program
• A game
• A mod (eg. Minecraft)
basically, anything that can be spooky EXCEPT hardware. This event is entirely software!
