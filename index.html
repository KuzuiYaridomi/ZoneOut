What ZoneOut actually is

ZoneOut is a Hack Club YSWS built for spooky software projects, ARG vibes, mysterious and thrilling weird creative ideas. Give people a great adrenaline rush, get back prizes! You do not need to be a pro coder to join!

What is Hack Club?

Hack Club

is a non-profit global community for teenagers who like building crazy projects. The whole point is to make, learn, share, and have fun while doing it! You can join our Slack community here!

What is a YSWS?

A YSWS; "You Ship, We Ship" You build something you're proud of, you ship the project. We'll ship back prizes that you'll like :p

What is ZoneOut?

ZoneOut is an event where you make something that gives users a big adrenaline rush. Your projects can be themed as horror, weird/eerie, thriller, mysterious, ARG and so much more! It's an event where you make stuff such as: websites, CLI tools, games, mods, and so much more.

Not a coder yet?

That is fine! Start small. Make a tiny website, a text adventure, a simple terminal project, or team up with friends. Everyone starts somewhere! search up a youtube tutorial, motivate yourself and you can surely do it. If you need help coding from the community, you can also hit us up in the code channel.

What should I make?

Anything eerie, creative, thrilling, or story-driven. If it feels like a strange artifact from another world, it probably fits. We want your projects to be weird and scare someone out!

What is the age requirement?

Hack Club events are open to ages 13-18 worldwide!

How do I track time?

You must use hackclub's own open-source time tracking tool:

Hackatime!
